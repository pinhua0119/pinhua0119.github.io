貳、 課程筆記

一、 Markdown 語法

1. 內文

(一) 直接輸入即可。
(二) 網頁文字換行時，程式碼需空一行。
Ex.

Hello World

我是阿柴

2. 標題

(一) 文字前方加上 # 。
(二) 標題分為一到六級，依照#數量表示，級數越小字體越小。
Ex.

(#) 一級標題

(##) 二級標題

(###) 三級標題

(####) 四級標題

(#####) 五級標題
(######) 六級標題

3. 粗體

(一) 在需要粗體的內文前後方各加上 ** 。
Ex.

我是阿柴，我喜歡學習。

4. 斜體

(一) 在需要斜體的內文前後方各加上 * 。
Ex.

我是阿柴，我喜歡學習。

5. 有序序列

(一) 在需要有序序列的內文前方加上數字和小數點並空一格即可(Ex. 1. ,2. )。
(二) 在有序序列中，即使前方數字相同，Markdown也會計算回正確的數字順序。
(三) 內縮一個有序序列，需要在前方打上四個空白鍵，若需要再內縮，則再打上四個空白鍵，以此類推。
Ex1.
  1. Hello World
    1. What is your name ?
      1. How are you ?
  2. 我是阿柴 (在程式碼中前方數字輸入為2)
Ex2.

  1. Hello World

  2. 我是阿柴 (在程式碼中前方數字輸入為1)

6. 無序序列

(一) 在需要無序序列的內文前方加上 * 並空一格即可。
(二) 內縮一個無序序列，需要在前方打上四個空白鍵，若需要再內縮，則再打上四個空白鍵，以此類推。
Ex.
  • Hello World
    • what is your name ?
  • 我是阿柴

7. 引用

(一) 在需要引用的內文前方加上 > 並空一格即可。
Ex.

Hello World

8. 超連結

(一) 外部連結 : [想要呈現的文字] (超連結)
(二) 內部連結 : [想要呈現的文字] (/blog/檔案名稱無須副檔名)
Ex.

pinhua0119 Github

9. 圖片

(一) 外部連結 : ! [] (圖片連結)
(二) 內部連結 : ! [] (/images/gallery/檔案名稱含副檔名)
Ex.

10. 單行程式碼

(一) 內文 小引號+程式碼+小引號
Ex.

我是阿柴，我喜歡學習。 print("Hello World")

11. 多行程式碼

(一) 多行程式碼無法打在內文中 [三個小引號(可輸入使用的程式語言)] +需換行+ [程式碼] +需換行+ [三個小引號]
Ex.
print("Hello World")
print("Hello World")
print("Hello World")

12. 表格

(一) 程式碼中第一列為類別，第二列虛線用於隔開類別與內容，第三列(含)以後為內容。
Ex.
Name Gender Age
Eric male 25
Rex male 23
Emily female 22

(一) {{< youtube 影片序號 >}}
Ex.

二、 HTML 語法

1. <div>、<a>、<img>

(一) <div> 通常用於包裹和分組 HTML 元素，方便對它們進行樣式設定和操作。
Ex.
<div>     <!-- <div> 標籤用來包裹一個標題和一段文字，將它們視為一個區塊 -->
  <h1>這是一個標題</h1>
  <p>這是一段文字。</p>
</div>
(二) <a> 用於創建鏈接到其他網頁或文件。
Ex.
<a href="https://www.google.com">Google</a>     <!-- <a> 標籤用來創建一個指向 Google 網站的鏈接，顯示的文字是 "Google" -->
(三) <img> 用於向網頁添加圖像。
<img src="https://example.com/image.jpg" alt="一張圖像">     <!-- <img> 標籤用來添加一個圖像到網頁中，使用 src 屬性指定圖像的 URL，使用 alt 屬性提供圖像的替代文字 -->

2. <div id="card"></div> 和 <div class="card"></div> 的差異。

(一) id 屬性值必須是唯一的，也就是說，頁面中不能有兩個元素使用相同的 id 屬性值。
(二) class 屬性用在 HTML 中元素指定一個或多個用空格分隔的類名，一個元素可以有多個類名。

3. Bootstrap Grid 功能。

(一) 一個網格系統，它提供了一種方便的方式來創建響應式網格佈局，可以輕鬆地實現不同大小螢幕的網頁設計。
Ex.
 <div class="col-sm-6 col-md-3">     <!-- 此例子中每列在小型螢幕上佔據 6 個列寬，在中型螢幕上佔據 3 個列寬。 -->

三、 CSS 語法

1. .card{} 和 #.card{} 的差異。

(一) .card{} ： 選取所有 class="card" 的 HTML 元素。
(二) #.card{} ：選取使用了 id="card" 的 HTML 元素。
Ex.在下面的例子中，.card{} 會選取兩個 <div> 元素，並為其設置背景顏色和 padding 屬性。而 #card{} 僅選取帶有 id="card" 的元素，並為其設置不同的背景顏色和 padding 屬性。
 <div class="card">這是一個使用 class="card" 的元素</div>
 <div id="card">這是一個使用 id="card" 的元素</div>

 .card {
  background-color: #F7CAC9;
  padding: 20px;
 }
 #card {
  background-color: #A6E7FF;
  padding: 20px;
 }

2. .card .card-title{} 代表的是甚麼意思 ?

(一) 表示一個層級關係，.card .card-title 表示 .card-title 是 .card 的子元素，並在 .card 的範圍內適用樣式。
